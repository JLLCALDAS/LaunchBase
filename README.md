# --> Desafio - 01
    -->Criar um programa para calcular o IMC de um determinada pessoa com a  seguinte formula: peso / (altura * altura) e verificasse se o mesmo estava ou não acima do peso mostrando no terminal o resultado utilizando template string.
    
    -->Criar um programa para calcular a aposentadoria de determinada pessoa verificando o sexo da mesma para efetuar o calculo correto e validar se estaria ou não apta para se aposentar.

# --> Desafio - 02
    -->Criar um objeto para armazenar os dados de uma empresa contendo um array para armazenar as informações ref. ao endereço da empresa e imprimir no console as informações utilizando template string.
    -->Criar um objeto para armazenar os dados de devs contendo um array para armazenar as techs de cada dev e imprimir no console os dados do dev e sua primeira opção como tech.

# --> Desafio - 03
    --> Criar um programa para ler um array de usuarios e mostrar cada usuario e suas techs.
    --> Criar um programa para buscar em um array de usuarios quais utilizava determinada tecnologia.
    --> Criar um programa para somar as despesas e as receitas de um array de clientes e listar o saldo de cada um indicando se o saldo era positivo ou negativo.
